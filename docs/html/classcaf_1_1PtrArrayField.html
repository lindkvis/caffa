Detailed Description

template<typename DataType>
class caf::PtrArrayField< DataType >

FieldClass to handle a collection of Object derived pointers The reasons for this class is to add itself as parentField into the objects being pointed to. The interface is made similar to std::vector<>, and the complexity of the methods is similar too.
