Detailed Description

Helper class for the Pointer class The add and removing of references is put into a pure static class to resolve circular include problems. Overall idea of the referencing system: The addressToObjectPointer is added to a std::set in the object pointed to. when the object pointed to is deleted, its destructor sets the object pointers it has addresses to nullptr
