template<typename T>
class caf::UiFieldSpecialization< Pointer< T > >

Partial specialization for Field< Pointer<T> >

Will package the Pointer<T> into Variant as Pointer<Object> Needed to support arbitrary types in Pointer without havning to declare everything Q_DECLARE_METATYPE() Also introduces the need for a isEqual() method, as this was the first custom type embedded in Variant
